# 🐱 CAT NET: CAT BREED INFORMATIONAL WEBSITE

## 📌 Overview:
- Cat Net is a multi-page informational website dedicated to exploring some of the world's most unique and fascinating cat breeds. The site provides detailed insights into the appearance, personality, and history of eight specific breeds. To enhance the user experience, the website features rich media galleries and custom-designed visual summaries.

## ✨ Key Features:
- **Breed Profiles:** Features in-depth pages for the Abyssinian, Japanese Bobtail, Munchkin, Norwegian Forest, Russian Blue, Savannah, Siamese, and Turkish Angora.
- **Custom Graphics:** Features a custom-designed Cat Net logo and unique "cat characteristics" icons created entirely from scratch using Figma. These icons serve as visual summaries for origin, lifespan, size, weight, coat pattern and length, playful nature, vocalness, affection, and price.
- **Rich Media Integration:** Includes extensive image galleries for every single breed. Additionally, embedded HTML5 video players are used to showcase the cats in action.
- **Intuitive Navigation:** Utilizes a persistent navigation bar with a dropdown menu. This allows users to easily jump between different breed profiles without returning to the home page.
- **Downloadable Content:** Each breed profile includes a link to print a PDF version of the page.
- **Attribution & Academic Integrity:** Contains a comprehensive references page meticulously documenting all external sources used for textual information and third-party images.

## 🛠️ Tech Stack:
- **Markup:** Semantic HTML5 to ensure accessible and well-organized content delivery.
- **Styling:** External CSS stylesheets linked directly to each individual page (index.css, abyssinian.css, etc.) to maintain modular and organized visual formatting.
- **Design:** Figma (used for creating the custom logo and infographic summary icons).
